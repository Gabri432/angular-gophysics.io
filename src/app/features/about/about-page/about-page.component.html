About gophysics

What is gophysics?

It is a collection of some of the most commonly used Physics formulas and constants.

The core idea was to provide a set of functions ready to be used to avoid making manual calculations.

It currently offers just a bit more than 60 formulas across different fields, such as classical physics, gravitational physics,relativity, thermodynamics, electromagnetism and fluids. While also other 50 constants from the same fields.

The library also provides some mathematical formulas and constants to further ease some calculations.

For the moment it isn't a stable library. That is, there are still a lot of functions that will be added.
This project is only in a beginning phase and may suffer several delays due to my exams and my university carrier.

DISCLAIMER: this project is NOT related to the following websites, companies or organitazions: thegophysics or perimeterinstitute.

Why using the Go programming language?

I started learning the Go programming language during my first academic year in the Programming 1 course.


This language is easy to use, statically typed, and provides a lot of cool features, such as being able to create functions with more than one output.
Infact almost all the functions in the library return two values, a float64 and a string.
Respectevely the value of the calculation and the unit of measurement.
